1. CSS Kya Hai?

CSS ka full form hai Cascading Style Sheets. CSS website ke visual design ko control karta hai. HTML batata hai page par kya content hai, CSS batata hai woh content kaise dikhega. Example: heading ka color, paragraph ka font size, button ka background, card ka shadow, section ka spacing, mobile layout, and hover effects sab CSS se control hote hain.

Real-world example: Agar HTML ek shop ka structure hai, toh CSS us shop ka paint, lights, shelves, signboard, and arrangement hai. Same product agar clean shop mein arranged ho toh customer easily samajhta hai. Website mein bhi CSS content ko professional and readable banata hai.

Types of CSS

Type Meaning Use Case
Inline CSS Style directly HTML tag ke andar likhte hain. Small testing ke liye, professional project mein avoid karna better.
Internal CSS HTML file ke head mein style tag ke andar CSS likhte hain. Single page demo ke liye useful.
External CSS Separate CSS file banakar HTML se connect karte hain. Professional projects ke liye best.

2. CSS File Ko HTML Se Connect Karna

External CSS use karne ke liye HTML file ke head section mein link tag add karna hota hai. Browser jab HTML read karta hai, toh link tag ke through CSS file load karta hai and styles apply karta hai.

<link rel="stylesheet" href="styles.css">

Line-by-Line Explanation

Code Detailed Explanation
<link> External resource connect karne ke liye use hota hai. CSS file, favicon, fonts jaise resources connect karne mein link tag useful hai.
rel="stylesheet" Browser ko batata hai ki linked file stylesheet hai. Matlab browser is file ko design rules ke form mein read karega.
href="styles.css" CSS file ka path hai. Agar HTML and CSS same folder mein hain toh direct file name likhte hain. File name mismatch hua toh CSS apply nahi hogi.

3. CSS Syntax

CSS syntax three main parts se milkar banta hai: selector, property, and value. Selector batata hai kis HTML element ko style karna hai. Property batati hai kaunsi style change karni hai. Value batati hai us property ki setting kya hogi.

h1 {
  color: blue;
  font-size: 36px;
}
Line Explanation
h1 Selector hai. Iska matlab page ke saare h1 headings par yeh CSS apply hogi.
{ Declaration block start karta hai. Iske andar CSS rules likhe jaate hain.
color: blue; Text color blue set karta hai. `color` property hai and `blue` value hai.
font-size: 36px; Heading ka font size 36 pixels set karta hai. Semicolon rule end karta hai.
} CSS declaration block close karta hai.

4. Selectors

CSS selectors decide karte hain ki style kis HTML element par apply hogi. Agar selector galat hai, CSS likhne ke baad bhi design apply nahi hoga. Isliye selector samajhna CSS ka foundation hai.

Element Selector

p, h1, body jaise selectors direct HTML tags ko target karte hain.

Class Selector

.card, .button jaise selectors class attribute wale elements ko target karte hain. Most reusable option.

ID Selector

#hero jaise selectors unique ID wale element ko target karte hain. ID page mein unique honi chahiye.

Pseudo-class

:hover jaise selectors special state target karte hain. Example: mouse hover hone par button color change.

p {
  color: #475569;
}

.section-title {
  color: #1d4ed8;
}

#main-heading {
  color: #0f172a;
}

.nav-links a:hover {
  color: #2563eb;
}

Code Explanation

Code Explanation
p Page ke saare paragraph tags ko target karta hai. Common text styling ke liye useful.
.section-title Class selector hai. Jis element par class section-title hogi, us par yeh style apply hogi.
#main-heading ID selector hai. Specific unique element target karta hai. ID ka use limited rakhna best hai.
.nav-links a:hover Nav links ke hover state ko target karta hai. Jab user mouse link par le jaata hai, tab color change hota hai.

5. Specificity

Specificity ka matlab hai CSS priority. Agar ek hi element par multiple CSS rules apply ho rahe hain, toh browser decide karta hai kaunsa rule stronger hai. Simple priority order: inline style, ID selector, class selector, element selector.

p {
  color: black;
}

.highlight {
  color: blue;
}

#special-text {
  color: red;
}

Agar ek paragraph par class highlight bhi hai and id special-text bhi hai, toh red color apply hoga because ID selector class selector se stronger hota hai.

6. Box Model and Layout Concepts

CSS box model har element ko ek rectangular box ki tarah treat karta hai. Har box ke andar content, padding, border, and margin hota hai. Content actual text/image area hai. Padding content ke andar ka space hai. Border box ke around line hai. Margin element ke bahar ka space hai.

.card {
  background: white;
  padding: 24px;
  border: 1px solid #e2e8f0;
  margin-bottom: 20px;
  border-radius: 8px;
}
Code Explanation
.card Card class wale elements ko target karta hai. Business website mein service cards ke liye use hoga.
padding: 24px; Card ke andar content ke around space add karta hai, text chipka hua nahi dikhta.
border Card ke around light line create karta hai, card boundary visible hoti hai.
margin-bottom Card ke bahar neeche space add karta hai, next element se distance banata hai.
border-radius Card corners rounded banata hai. Modern UI mein soft corners professional lagte hain.

7. Flexbox Layout System

Flexbox one-dimensional layout system hai. Iska matlab flexbox ek direction mein layout control karne ke liye best hai: row ya column. Navbar, button groups, hero actions, and alignment ke liye flexbox bahut useful hai.

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 24px;
}
Code Explanation
display: flex; Parent container ko flex layout bana deta hai. Child elements row mein arrange ho jaate hain by default.
justify-content: space-between; Horizontal direction mein first item left and last item right side push hota hai. Logo left, nav right.
align-items: center; Items vertically center align hote hain. Navbar clean and balanced dikhta hai.
gap: 24px; Flex items ke beech consistent spacing add karta hai.

8. CSS Grid Layout System

CSS Grid two-dimensional layout system hai. Grid rows and columns dono manage kar sakta hai. Services section, gallery, pricing cards, dashboard, and page layout ke liye grid very powerful hai.

.services-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 24px;
}
Code Explanation
display: grid; Container ko grid layout mein convert karta hai.
grid-template-columns Grid ke columns define karta hai.
repeat(3, 1fr) 3 equal columns create karta hai. `1fr` available space ka equal fraction hota hai.
gap: 24px; Cards ke rows and columns ke beech spacing add karta hai.

9. Responsive Design Principles

Responsive design ka matlab website desktop, tablet, and mobile sab par properly dikhe. Responsive website mein text readable hota hai, images overflow nahi karti, cards mobile par stack ho jaate hain, and user ko horizontal scroll nahi milta.

img {
  max-width: 100%;
  height: auto;
}

.container {
  width: min(1120px, 92%);
  margin: 0 auto;
}
Code Explanation
max-width: 100%; Image parent container se bahar nahi niklegi. Mobile par image shrink ho sakti hai.
height: auto; Image ratio maintain karta hai. Image stretch nahi hoti.
width: min(1120px, 92%); Desktop par max 1120px width, small screen par 92% width. Yeh responsive container pattern hai.
margin: 0 auto; Container ko center align karta hai.

10. Media Queries

Media query screen size ke according CSS change karne ka method hai. Example: desktop par 3 cards ek row mein, mobile par cards one-by-one. Media queries responsive design ka core part hain.

@media (max-width: 768px) {
  .navbar {
    flex-direction: column;
  }

  .services-grid {
    grid-template-columns: 1fr;
  }
}
Code Explanation
@media (max-width: 768px) Yeh CSS sirf tab apply hogi jab screen width 768px ya usse kam hogi.
flex-direction: column; Navbar items row se column mein aa jaate hain. Mobile menu readable hota hai.
grid-template-columns: 1fr; Services grid mobile par one-column layout ban jaati hai.

11. Animations and Transitions

Transition style change ko smooth banata hai. Example: button hover par color suddenly change hone ke bajay smoothly change ho. Animation keyframes ke through repeated ya staged movement create karta hai. Module 3 mein hum simple hover transitions use karenge.

.button {
  transition: transform 0.2s ease, background 0.2s ease;
}

.button:hover {
  transform: translateY(-2px);
  background: #1d4ed8;
}
Code Explanation
transition Hover par transform and background change smooth bana deta hai.
.button:hover Button par mouse hover hone par yeh style apply hota hai.
translateY(-2px) Button ko thoda upar move karta hai. Interactive feel aata hai.

12. CSS Performance and Best Practices

Good CSS sirf beautiful nahi, maintainable bhi hota hai. Agar CSS random likha gaya, future mein change karna difficult ho jaata hai. Professional CSS mein reusable classes, consistent spacing, limited colors, clean naming, and responsive testing important hote hain.

  • External CSS file use karo.
  • Class names meaningful rakho.
  • Repeated styles ke liye reusable classes banao.
  • Too many colors and fonts avoid karo.
  • Mobile view hamesha test karo.
  • Unused CSS remove karo.
  • Specificity unnecessarily high mat karo.
